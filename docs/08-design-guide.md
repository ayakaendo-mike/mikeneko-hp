# デザインガイド（たたき台）

> ⚠️ このドキュメントは仮案です。ロゴ・コーポレートカラー・指定フォント・猫モチーフ素材が未確定のため（[01-requirements.md](./01-requirements.md) 未確定事項参照）、確定次第このドキュメントを更新する。

## デザイン方針（Project.md記載事項）
- シンプル
- 余白を活かす
- 信頼感
- 親しみ
- AI企業らしい先進性
- 猫モチーフを少し取り入れる
- 過度な装飾はしない
- アニメーションは上品に

## 現時点での方向性（案）
- **トーン**: 「伴走するデザイン会社」のコンセプトに合わせ、堅すぎず・軽すぎない中間的な信頼感を演出
- **カラー**: 実際のブランドカラーが無い場合、信頼感を表すネイビー/インディゴ系をベースに、猫モチーフを連想させる差し色（例: 温かみのあるアクセントカラー）を1色加える方向を仮提案。確定はブランド素材次第
- **フォント**: 和文はゴシック体ベースで可読性重視。指定フォントがあれば差し替え
- **猫モチーフ**: オリジナルキャラクターの有無が未確認。無い場合は控えめなアイコン/イラスト程度の使用に留め、過剰な装飾を避ける
- **アニメーション**: ページ遷移・スクロール時のフェード等、上品で控えめな範囲に留める（派手なパララックス等は避ける）

## 未確定のため要確認
- ロゴデータ（ai/svg等）の有無
- コーポレートカラーのカラーコード指定の有無
- 指定フォントの有無
- 猫モチーフの具体的な素材（既存イラスト/キャラクターがあるか、新規制作が必要か）
- 使用可能な写真素材（オリジナル撮影 or ストックフォト）

これらが揃い次第、コンポーネント単位のデザインシステム（カラートークン、タイポグラフィスケール、余白ルール等）に落とし込む。
