# 要件定義

## 1. 確定事項サマリー

| 項目 | 内容 |
|---|---|
| 構築種別 | 完全新規構築（既存サイト・URL資産なし） |
| 技術スタック | Astro（コード・構造）+ microCMS（コンテンツ） ハイブリッド構成 |
| ホスティング | Xserver（ドメイン取得済み・サーバー未契約） |
| 運用体制 | 社内スタッフがAIを使いながらコンテンツ更新／Claude Codeがコード・構造を改善 |
| リポジトリ | GitHub（Private推奨） |

詳細な決定理由・経緯は [decisions.md](./decisions.md) を参照。

## 2. ページ一覧
確定サイトマップは [02-site-structure.md](./02-site-structure.md) を参照。

## 3. 機能要件

- レスポンシブ対応（モバイルファースト）
- お問い合わせフォーム（送信先・自動返信・スパム対策は未確定 → 7章参照）
- OGP設定（全ページ）
- パンくずリスト（構造化データ対応）
- 構造化データ: LocalBusiness / BreadcrumbList / Article / FAQPage / Service
- ブログ（カテゴリ・タグ対応）
- お知らせ
- XML Sitemap自動生成
- robots.txt
- 404ページ
- リダイレクト対応（将来のURL変更に備える）
- サイト内検索（将来実装・Pagefind想定、初期リリース対象外）

## 4. 非機能要件

| 分類 | 目標 |
|---|---|
| パフォーマンス | Core Web Vitals: LCP < 2.5s / INP < 200ms / CLS < 0.1 |
| アクセシビリティ | WCAG 2.1 AA準拠を目標 |
| セキュリティ | 静的サイト構成による攻撃面の最小化。フォームのスパム対策必須（reCAPTCHA等） |
| 保守性 | コード変更はPRベース。コンテンツ更新はGit知識不要（microCMS経由） |
| SEO | 全ページにtitle/meta/OGP設定。構造化データ実装。Search Console連携 |

- [x] 制作実績：掲載可能件数、守秘義務の制約有無 → Web制作4〜5件／グラフィックデザイン4〜5点／名刺制作5〜6点／サポート事例2件（[decisions.md#d006](./decisions.md#d006)）
- [x] お客様の声・レビューの収集有無 → **現時点で無し**。`/voice/`ページは初期リリースでは非掲載とし、将来収集でき次第追加する（[02-site-structure.md](./02-site-structure.md)）
- [x] 公開目標日・予算感 → **2026年7月中に公開**。予算はサーバー代＋AI利用料程度に抑えたい（低予算方針）

## 8. 料金ページ（仮掲載・ブラッシュアップ前提）

営業資料（[営業資料/みけねこデザイン_会社案内.html](../営業資料/みけねこデザイン_会社案内.html)）に掲載していた価格帯をそのまま初期版として転記する。金額は「目安」であることを明記し、正確な見積もりはヒアリング後とする。

| サービス | 価格帯目安 |
|---|---|
| 名刺・チラシなど（グラフィックデザイン各種） | 数万円〜 |
| ランディングページ（1ページ完結型のLP） | 20〜50万円程度 |
| コーポレートサイト（会社の顔となるホームページ） | 30〜100万円程度 |
| リニューアル（既存サイトの改善・作り直し） | 内容に応じてご提案 |
| 保守・運用（更新代行・改善サポート） | 月額制にも対応 |

注記: 「ご要望・ページ数・機能によって変動します。ヒアリングのうえ、最適なお見積りをご提案いたします。」／「スモールスタートOK」「段階的な拡張プラン」「補助金活用のご相談も」を併記する。

## 9. 予算・スケジュール方針

- **公開目標**: 2026年7月中
- **予算方針**: サーバー代（Xserver）とAI利用料程度に抑える。有償の外部サービス（microCMS有料プラン、CDN、フォームSaaS等）はできる限り無料枠内に収める
- **影響**: この方針により、Xserverプラン・microCMSプランは共に「無料または最安で要件を満たせるか」を必ず事前検証する（[04-tech-architecture.md](./04-tech-architecture.md) 参照）。有料化が避けられない場合は、その旨を事前に共有し判断を仰ぐ
