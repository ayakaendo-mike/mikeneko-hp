# プロジェクト概要

## プロジェクト名
みけねこデザイン コーポレートサイト制作

## 位置づけ
本サイトは単なる会社紹介ページではなく、以下5つの役割を同時に持つ事業資産とする。

1. **営業ツール** — 名刺交換後・商談時の信頼獲得
2. **ブランディング** — 「伴走するデザイン会社」というポジショニングの体現
3. **SEO資産** — 地域×サービスキーワードでの継続的な問い合わせ獲得
4. **AI運用実験場** — Claude Code等のAIエージェントによる開発・改善の実践の場
5. **将来的なAI社員による運用対象** — 人間の関与を段階的に減らしながら自律運用へ移行する土台

## 目的

### 主目的
- 名刺交換後に会社を知ってもらう
- 商談時の信頼獲得
- サービス内容を分かりやすく伝える
- 実績を紹介する
- 問い合わせにつなげる

### 中長期目的
SEOによる集客。以下キーワード×地域の掛け合わせで地域検索からの問い合わせ獲得を狙う。

- **対象キーワード**: ホームページ制作／WordPress制作／AI活用支援／DX支援／LINE構築／ITサポート
- **対象地域**: 松戸市／鎌ケ谷市／船橋市／市川市／千葉県

## コンセプト
**「伴走するデザイン会社」**
デザインだけでなく、IT・AI・DXまで相談できるパートナーになる。

## ターゲット
中小企業／個人事業主／女性起業家／地域企業／商工会会員／建設業／製造業／士業

## デザイン方針
シンプル／余白を活かす／信頼感／親しみ／AI企業らしい先進性／猫モチーフを少し取り入れる／過度な装飾はしない／アニメーションは上品に

（具体的なカラーコード・フォント・猫モチーフ素材は [08-design-guide.md](./08-design-guide.md) で管理。現時点では素材待ちの箇所あり）

## ゴール
みけねこデザインを代表するブランドサイトを構築し、サイトそのものが営業マンとなり信頼を生み、問い合わせを獲得し、AI運用の実験場として成長し続けること。

## KPI（案・要確認）
Project.mdに数値目標の記載が無いため、以下を暫定提案。運用開始後に実数値へ更新する。

| 指標 | 目標(案) |
|---|---|
| 月間オーガニック流入 | 3ヶ月目: 300セッション／6ヶ月目: 1,000セッション |
| 問い合わせ件数 | 月2件 → 月5件（6ヶ月目） |
| Core Web Vitals | 全指標「良好」判定を維持 |
| インデックス済みブログ記事数 | 6ヶ月で24本（月2〜4本ペース） |

> ⚠️ この数値は暫定案です。実際の集客目標・商談化率を踏まえてご確認・修正ください。
